# 🧳 Travel Recommendation AI

> **Классическое машинное обучение** для персонализированных рекомендаций мест для путешествий на основе оценок, бюджета и сезона.


## 📌 О проекте

Проект демонстрирует создание **рекомендательной системы для туристических мест** с использованием классических алгоритмов машинного обучения (без глубокого обучения).

**Признаки:**
- ⭐ **Рейтинг места** (от 1.0 до 5.0)
- 💰 **Бюджет поездки** (USD)
- 🌤️ **Сезон** (зима, весна, лето, осень)

**Целевая переменная:**  
`recommend` – рекомендуется ли место для поездки (1 – да, 0 – нет).

Генерация синтетического датасета (10 000 записей) основана на логических правилах и небольшом шуме, имитирующем реальные данные.

---

## 🧠 Использованные модели машинного обучения

В проекте обучены и сравнены **4 классических алгоритма**:

| Модель | Тип | Особенность |
|--------|-----|--------------|
| Logistic Regression | Линейная | Базовый классификатор, интерпретируемый |
| Random Forest | Ансамбль деревьев | Устойчив к переобучению, даёт важность признаков |
| SVM (RBF kernel) | Метод опорных векторов | Хорошо работает на небольших размерах выборки |
| Gradient Boosting | Бустинг | Высокая точность, но медленнее обучается |

**Выбор оптимальной модели** производился по метрике **F1-score** (баланс точности и полноты).  
Дополнительно анализировались: accuracy, precision, recall, ROC-AUC, матрица ошибок и кривые обучения.

> 🏆 **Лучшая модель:** `Random Forest` (F1 ≈ 0.95, ROC-AUC ≈ 0.98)
